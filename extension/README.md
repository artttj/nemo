# Nemo Vault - Browser Extension

A secure local password vault with brutalist, functional design.

## Loading the Extension

1. Open Chrome/Edge and navigate to `chrome://extensions/`
2. Enable "Developer mode" (toggle in top right)
3. Click "Load unpacked"
4. Select this `extension/` directory

## Features

- **Vault locked**: Compact locked screen with feature tokens
- **Credential list**: Structured list with search and filters
- **Copy to clipboard**: Click copy buttons to copy credentials
- **Local storage**: All data stored locally in browser storage
- **Responsive**: Designed for extension popup (360px width)

## Usage

1. Click "Unlock vault" to access credentials
2. Click "New credential" to add sample items
3. Search by title, username, or URL
4. Filter by type (All, Logins, Cards, Notes)
5. Click items to expand and see details
6. Use copy buttons to copy values

## Icons

Icon placeholders are included. For production:
